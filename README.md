# Decotechs_Task
This is simple expense tracker created with MERN

Dependencies
-----------
1: Node
2: MongoDB


To run backend, enter following terminal commands in backend directory 
----------------------------------------
1: npm install
2: npm start


To run frontend, enter following terminal commands in frontend directory
----------------------------------------
1: npm install
2: npm start
